sx_status_t sx_api_issu_pause_set ( const sx_api_handle_t  handle,
const sx_issu_pause_t *  pause_params_p 
)

This API prepares the SDK for CRIU (checkpoint/restore in userspace) Hibernation. The SDK configuration is written to persistent storage from where it can be restored. This is also referred to as SDK "Warm Boot" Mode Supported devices: Spectrum, Spectrum 2, Spectrum 3.

Parameters
[in]handle- SX-API handle.
[in]pause_params_p- pause params.
Returns
sx_status_t:
SX_STATUS_SUCCESS - Operation completes successfully
SX_STATUS_PARAM_ERROR - Input parameters error - NULL handle
SX_STATUS_COMM_ERROR - Input parameters error - Invalid handle
SX_STATUS_CMD_UNPERMITTED - if SDK is already paused
SX_STATUS_ISSU_IN_PROGRESS - When ISSU is already in progress.
SX_STATUS_NO_MEMORY - Out of Memory
